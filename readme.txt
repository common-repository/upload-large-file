Increase file Upload size limit. 


== Description==
Increase any file Upload size limit through dashboard directly.
Plugin automatically detects upload limits set by WordPress and by the server / hosting provider, and displays them.


== Installation ==

* Upload the Upload Large File plugin to the `/wp-content/plugins/` directory
* Activate the Upload Large File plugin through the 'Plugins' menu in WordPress
* You can use it from WordPress menu 'Upload Large File' to change upload size.


== Frequently Asked Questions ==

= How to use plugin? =
* After activating plugin, go to WordPress dashboard and you will see menu on left 'Upload Large File'.

= Does this plugin work with all servers and hosting providers? =
* Yes, it works with all servers and hosting providers. However, please know that server adjusted limits can't be changed from a WordPress plugin. If server limit is lower then you have to email your hosting provider and ask them to increase the limit.
